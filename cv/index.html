    • Publications
    		Adaptive Neural Network Control of an Airborne Robotic Manipulator System

    Halbert Xu, Shuzhi Sam Ge, Qiong Liu, Wanyue Jiang, Rihang Ji

    The 2nd International Conference on Industrial Artificial Intelligence (ICIAI), 2020

    In this paper, adaptive neural network control is studied for an Airborne Robotic Manipulator (ARM) system. To handle the uncertainties and disturbances of the ARM system and improve its robustness, radial basis function neural network (RBFNN) is used for approximating unknown dynamics model of the system to realize better adaptive neural network control. The proposed RBFNN-based control scheme is used for approximating errors, which can be effective in making learning objective smaller and learning time shorter compared with conventional approaches.

    		A Simpler Adaptive Neural Network Tracking Control of Robot Manipulators by Output Feedback

    Qiong Liu, Shuzhi Sam Ge, Yan Li, Mingye Yang, Halbert Xu, Keng Peng Tee

    The 6th International Conference on Control, Automation and Robotics (ICCAR), 2020

    The trajectory tracking problem of the manipulator is investigated by a simpler design adaptive neural network (NN) in this paper. The output feedback control is considered when the velocity information cannot be obtained. The weights of Radial Basis Function NN converge to its optimal value by using the auxiliary filter to estimate weights error.
