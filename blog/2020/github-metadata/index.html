Biography

September 28, 2020

籍贯中国安徽省庐江县， 1999年入安徽省淮南市潘集区第一小学，2005年考入安徽省洞山中学学习，2008年考入安徽省淮南第二中学学习 2011年，17岁考入东北大学，攻读机械工程及自动化专业。2014年夏，前往天津大学参加学术夏令营 2015年6月大学毕业后于翌月前往江苏省镇江市镇江华东电力设备制造厂工作， 初在国营工厂车间工作。2016年8月通过了转正定级并被分配至母线产品设计部任设计工程师。 2017年8月开始筹备出国留学事宜，周一至周五忙于发电站设计工作，利用周六和周日时间赶早上7点多的火车至南京新街口参加出国英语学习班，这种忙碌的状态持续了半年多。 2018年11月收到了新加坡国立大学的硕士研究生录取通知书，旋即筹备赴南洋求学事宜。2019年1月3日第一次到达新加坡。 2019年夏，第一次回国前往广东省深圳市南方科技大学参加学术夏令营，此次回国十天，途经了广州、武汉等地。 2020年1月硕士研究生毕业，在电子与计算机工程系担任助教和研究助理工作 。2020年4月初，新加坡封城，辗转于6月15日到达上海
